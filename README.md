# MiniBank-ConsoleApp
Консольное приложение на C#, имитирующее работу банковского терминала.

### Реализованный функционал:
- Имитация авторизации пользователя по номеру аккаунта и ПИН-коду.
- Проверка баланса, пополнение счёта и снятие наличных с валидацией данных.
- Логирование истории операций с указанием даты и точного времени.
- Сохранение баланса и истории транзакций в файловую систему (данные не пропадают после перезапуска приложения).
